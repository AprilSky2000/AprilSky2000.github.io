1. linux基础

1.1 命令行模式

  1. 登录

    成功后进入

    注意：一般身份用户的提示字符如上，为$；root的提示字符为#

    注销Linux

  2. 模式切换
    Linux默认提供六个终端给用户登录，分别为 tty1 - tty6的操作接口环境

    • Ctrl + Alt + F2 ~ F6：命令行模式登录 tty2 - tty6终端
    • Ctrl + Alt + F1：图形用户界面模式

  3. 热键

    按键 执行结果
    [Tab] 命令或文件名的补全
    [Ctrl]+c 终止当前命令
    [Ctrl]+d 输入结束(EOF)
    [Ctrl]+m 回车
    [Ctrl]+s 暂停屏幕输出
    [Ctrl]+q 恢复屏幕输出
    [Ctrl]+u 在提示字符下，将整列命令删除
    [Ctrl]+z 暂停当前命令

1.2 绝对路径与相对路径

  • 绝对路径：有根目录（/）开始写起
  • 相对路径，开头不是/就属于相对路径写法

1.3 命令的查找顺序

  1. 以相对/绝对路径执行命令，如【/bin/ls】或【./ls】
  2. 由【alias】找到该命令来执行
  3. 由bash内置的（builtin）命令来执行
  4. 通过 $PATH变量内的顺序查找到第一个命令执行

1.4 ANSI转义序列

在命令行中对字符属性进行改变需要依靠转义序列

  1. \\033：转移序列是以 ESC开头的，ESC在ASCII码中即为$(27)_{10}$或$(33)_8$，在shell中表示为 ^[
  2. [：说明接下来开始定义文本属性和颜色属性
  3. 文本属性定义如下：| 字体代码 | 字体说明 |
    | ——– | ——– |
    | 0 | 默认值 |
    | 1 | 粗体 |
    | 22 | 非粗体 |
    | 4 | 下划线 |
    | 24 | 非下划线 |
    | 5 | 闪烁 |
    | 25 | 非闪烁 |
    | 7 | 背景色 |
    | 27 | 非背景色 |
  4. 颜色属性定义如下：分为前景色和背景色，前景色第1个数为3，背景色第1个数为4| 前景色/背景色代码 | 颜色说明 |
    | —————– | ——– |
    | 30/40 | 黑色 |
    | 31/41 | 红色 |
    | 32/42 | 绿色 |
    | 33/43 | 黄色 |
    | 34/44 | 蓝色 |
    | 35/45 | 洋红 |
    | 36/46 | 青色 |
    | 37/47 | 白色 |
  5. 关闭转义序列：\\033[0m，也是shell的默认色
