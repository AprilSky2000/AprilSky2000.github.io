Vim

Author: April Sky Date: November 20, 2023  10:25:45 Category: Tool

1. vim的三种模式

  • 一般命令模式：vi打开一个文件直接进入该模式，可进行复制、粘贴、删除
  • 编辑模式：可以编辑文件内容
  • 命令行模式：输入 :、/、?三个中任意一个，可将光标移到最下面一行

2. 一般命令模式按键

2.1 移动光标

按键 作用
0 移动到这一行最前面字符
$ 移动到这一行最后面字符
G 移动到文件最后一行
gg 移动到文件第一行

2.2 查找

按键 作用
/word 光标之下查找名为word的字符串
?word 光标之上查找名为word的字符串
n 重复前一次查找
N 对n进行反向操作

2.3 替换

2.4 删除、复制、粘贴

删除
x或X 删除一个字符
dd 删除整行
ndd n为数字，删除光标向下n行
复制/粘贴
yy 复制光标所在那一行
nyy n为数字，复制光标向下n行
p或P p将已复制内容在光标下一行粘贴，P在光标上一行粘贴
u 撤销上一次操作

3. 编辑模式

i/a/o均为进入插入模式，[Esc]为退出插入模式

4. 命令行模式

命令
:w 编辑数据写入文件
:q 退出vi
:! command 暂时退出vi，到命令行模式执行command的显示结果
:set nu 显示行号
:set nonu 取消行号
:set nowrap 将文本碰到右边自动换行取消，只有文本中有换行符时才换行

如果加上 !，代表强制写入或强制退出

5. 可视模式

可以高亮选中一部分文本

  • v命令进入字符可视模式，文本选择是以字符为单位的
  • V命令进入行可视模式，文本选择是以行为单位的
  • [Ctrl+V]/[Ctrl+Q]进入块可视化模式

6. vim环境

整体vim设置一般放在 /etc/vimrc文件中，这个文件建议不动，在 ~/.vimrc（默认不存在）这个文件中修改设置
